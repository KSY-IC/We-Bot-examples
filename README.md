## 必要な環境
### Python
本サンプルはPython3.7以降で動作確認を行っています。
使用する前に、We-BotのPythonライブラリをインストールしてください。

### Scratch
本サンプルはScratch3を用いて動作確認を行っています。
使用する前にScratch3のオフラインエディタをインストールしてください。
